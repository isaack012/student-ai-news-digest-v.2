# Student Tech/AI News Digest

## Overview
This is an AI-powered application that researches, summarizes, and synthesizes information from multiple sources to create a student-friendly news digest.

## Features
- Takes a topic and multiple article URLs
- Extracts and summarizes each article
- Compares sources and identifies trends
- Generates a final newsletter-style output for students

## Tech Stack
- Python
- Streamlit
- OpenAI API
- newspaper4k

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Description
This AI agent goes beyond simple summarization by comparing multiple sources, identifying trends, and generating structured content tailored for students.
